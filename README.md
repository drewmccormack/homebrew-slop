# homebrew-slop

Homebrew tap for [slop](https://github.com/drewmccormack/slop) — AI slop comes to the command line.

    brew install drewmccormack/slop/slop

Builds from source. Requires an Apple-Silicon Mac on macOS 26+ with the Xcode
Command Line Tools (`xcode-select --install`) and Apple Intelligence enabled.
